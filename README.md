Libash
======

Collection of scripts that will make your bash experience much more pleasant.



Usage
-----

Simply clone this repo, and source `libash` in your `.bashrc`.



Stuff included
--------------

- colorful progressbar
- b/w progressbar
- right prompt (to come)
- chpwd hook
